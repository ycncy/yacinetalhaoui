Jeu automate cellulaire : "FireFighter"

Réalisé avec :

Projet réalisé dans le cadre de l’UE Programmation et Conception (6 ETC).

Objectif : Créer une application Java en respectant les principes SOLID.

Sujet : Implémenter un proto-simulateur d'automate cellulaire. On commencera par une execution sur une grille où les éléments seront des pompiers, des pompiers motorisés, des feux, des montagnes et des nuages.
À chaque tour :

  • Pompier éteindra soit tous les feux autour de lui soit se dirigera vers le feu le plus proche et éteindra ensuite le feu autour de lui.
  • Le feu s'étendra autour de lui tous les deux tours.
  • Le pompier motorisé se comporte comme un pompier mais se déplace de 2 cases au lieu d'une
  • Le nuages se déplacent aléatoirement dans la grille en éteignant le feu autour de lui
